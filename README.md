# Instructions

    1. Go to https://repl.it/
    2. Create an account using the **Sign up** button or log in to your account using the **Log in** button.
    3. Click on the **Import code or design** button.
    4. Choose the **GitHub** button.
    5. Copy and paste *https://github.com/stephencattaneo/scouting-mb-python* into the text box and click the **Import from GitHub** button.
    6. Wait a moment for repl.it to import the code from the GitHub repository.
    7. Explore the code and comments.
    8. In the AI box, ask it to run your program. This will set up the environment required to run the code you're looking at.
    9. Click the green > button to run the code to see how the program operates.
    10. Click the Console tab to interact with your program.
    11. Make 4 of the 7 changes suggested at the top of the file
    12. Repeat steps 3 - 11 for the following projects:
        1. **JavaScript** - https://github.com/stephencattaneo/scouting-mb-javascript
        2. **Golang** - https://github.com/stephencattaneo/scouting-mb-golang

##  Coding Project Troubleshooting
If something breaks, ask the AI assistant for help.
If something doesn't make sense to you, ask AI to explain it to you.

Remember that all AI has been trained on content from the internet. If you're not satisfied with what you're getting from AI, other trustworthy sources of information are redit.com and stackoverflow.com.
